# Bank Balances

The `query balance` command will return all the balances in the bank module for the given wallet.

```sh
hanchond playground query balance evmos1dh0j0cppe0uktzl8reukddztkcygzrh7t5fx0n --node 2
balances:
- amount: "3"
  denom: ibc/8EAC8061F4499F03D2D1419A3E73D346289AE9DB89CAB1486B72539572B1915E
pagination:
  next_key: null
  total: "0"

```
