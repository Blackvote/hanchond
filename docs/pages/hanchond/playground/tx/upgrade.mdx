# Upgrade Proposal

The command `tx upgrade-proposal` can be used to create an upgrade proposal.

It requires the tag/version name and the upgrade height, it returns the transaction hash.

```sh
hanchond playground tx upgrade-proposal v19.0.0 200000
8F4FBF03B147607B80A778928FFBCC196CAABEA67E98DA64745280D29EE34795
```
