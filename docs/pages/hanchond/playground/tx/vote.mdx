# Vote

The command `tx vote` votes in all the active proposals with the given option.

It returns a list of all the transaction hashes.

```sh
hanchond playground tx vote
9931109EBFC178158A93999F0E3A4EF889136D63A9568F3D0512E3796D60584F
```

:::info
The flag `--option` can be used to change the voting option, it defaults to `yes`
:::
