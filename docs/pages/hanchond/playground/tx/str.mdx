# STRv1 Proposal

## Register Coin Proposal

The required argument is the token denomination:

```sh
hanchond playground tx str-v1-proposal ibc/8EAC8061F4499F03D2D1419A3E73D346289AE9DB89CAB1486B72539572B1915E --node 2
6ED735C25684DA7457E9A86DB4C8684A2C7A7D155CDF8E20CD8782B1E6337C8D
```

There are four available flags:

- exponent: defaults to `18`
- alias: defaults to `tokenalias`
- name: defaults to `tokenname`
- symbol: defaults to `tokensymbol`
