# Start node

After building the version and creating the genesis file, the node can be started using the `start-node` command

```sh
hanchond playground start-node 1
Evmos is running with id: 95753
```

:::info
The node logs can be found inside the home folder `data/${id}/run.log`, by default it is `~/.hanchond/data/${id}/run.log`
:::
